### - day02(220505,목) : 수열02
- <문제> : 1-2+3-4+ ... +99-100의 합계
- 해결 완료
  - 확장 문제1 : 1-2+3-4+5-6+ ... +99 -> 해결
  - 확장 문제2 : (-1)×2×(-3)×4×(-5)× ··· ×100  -> 해결
- 배운점
  - 문자열 f-formatting 확실히 습득
  - 서식 지정자 자료형 확인 : https://dojang.io/mod/page/view.php?id=2305
- <소회> : 차근차근, 확실히
